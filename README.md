# Landing Page - Papelaria UX

> **Descrição:** Uma landing page moderna e responsiva desenvolvida para a **Papelaria UX**, focada em apresentar produtos de forma intuitiva e elegante, proporcionando uma excelente experiência de navegação para o usuário.

---

## Objetivo do Sistema
O objetivo principal deste projeto é criar uma vitrine digital eficiente que permita:
*   Apresentar o catálogo de produtos de forma organizada.
*   Fortalecer a presença digital da marca.
*   Garantir uma interface amigável (User Experience) para conversão de novos clientes.

---

## Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web para garantir leveza e performance:

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização, layout (Flexbox/Grid) e design responsivo.
*   **JavaScript:** Implementação de interatividade e dinamismo na página.
